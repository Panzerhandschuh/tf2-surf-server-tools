Default Cvars:

"tf2t_enable" = "1"
- Enable/Disable TF2T

"tf2t_speedometer" = "1"
- Default Speedometer value

"tf2t_scoutboostamount" = "1.2"
- Scout boost modifier (1.2 = normal TF2T, 1.0 No change)

"tf2t_highjump" = "1"
- Enable/Disable Highjump


Client Commands:

!speedometer/!turbo
- Client chat command to enable/disable speedometer

sm_speedometer/sm_turbo
- Client console command to enable/disable speedometer
