Default Cvars:

"sm_wpnblock" = "1"
 - Wpnblock type: 0 = disable, 1 = remove on spawn (only), 2 = repeatable removal of weapons, see sm_wpntime.

sm_wpnblock_add 
 - Add a blocked weapon. 

sm_wpnblock_clear 
 - Clear list of blocked weapons.

sm_wpnblock_list
 - Show blocked weapons.

sm_wpnblock_version 
 - WpnBlock version.

"sm_wpntext" = "1"
 - Enable printing of which weapons which has been disabled.

"sm_wpntime" = "5.0"
 - Time to remove weapons again if sm_wpnblock = 2.
